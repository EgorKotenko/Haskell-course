1. Записать следующую функцию в point-free стиле: func x l = map (\y -> y*x) l. В комментариях показать все промежуточные шаги преобразований.

2. Реализовать функцию, генерирующую бесконечный список чисел Фибоначчи. Функция должна быть реализована с помощью "завязывания в узел". Возможно, будет полезной функция zipWith.

3. Вывести первую позицию в списке, на которой сумма двух соседних элементов максимальна. Например, для [1, 5, 6, 2] функция должно вернуть значение 2.

4. Посчитать высоту дерева.

5. Посчитать минимальное расстояние в дереве от корня до листьев.

6. Реализовать функцию, которая по произвольной строке проверяет корректность скобочной последовательности в этой строке.
